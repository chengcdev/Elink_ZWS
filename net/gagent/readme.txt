  @par Demo Description 
This demo shows:  
    - how to run gagent sdk on MiCO system. 
      Include the following specifical function:
       - how to use mico gagent sdk;
       - how to add a sample application using mico gagent sdk.

@par Directory contents 
    - gagent/samples/mico_main.c    Applicaiton program
    - gagent/samples/mico_config.h   MiCO function header file
    - gagent/samples/user_defines.h     Application defines header file
    - gagent/samples/user_delegate.c   Application functions && sdk callbacks by user.

@par Hardware and Software environment  
    - Using MiCOKit with MXCHIP EMW3165/3162/3238 Wi-Fi modules to run the sample application.
    - Using ST-LINK/JLINK to debug the sample code.
    - This demo can be easily tailored to any other supported device and development board.

@par How to use it ? 
In order to make the program work, you must do the following :
 - Open your preferred toolchain, 
    - IDE:  IAR 7.30.4 or Keil MDK 5.13.       
    - Debugging Tools: JLINK or STLINK
 - Run the sample application: 
    - Please referring to: docs\ReleaseNote_MiCO_GAgent_SDK.
